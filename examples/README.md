# Examples

各エージェントの使用例とサンプルプロジェクト

## サンプルプロジェクト

実際の使用例を示すサンプルプロジェクトを提供します。

### 含まれる予定のサンプル

- **Memory Safety**: メモリリークのある/ないC++コードの比較
- **Interop**: C#からC++ライブラリを呼び出す実例
- **Build**: CMakeを使ったマルチプラットフォームプロジェクト
- **Windows Desktop**: WPF MVVMアプリケーション
- **Performance**: 最適化前/後のパフォーマンス比較
- **Testing**: 完全なテストスイートの例
