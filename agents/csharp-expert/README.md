# C# Expert Agent

.NET 8+とモダンC#開発のエキスパートエージェント

## 概要

このエージェントは、最新の.NET技術とC#のベストプラクティスを適用し、エンタープライズグレードのアプリケーション開発を支援します。

## 主な機能

### モダンC#機能
- Async/await patterns
- LINQ and expression trees
- Pattern matching
- Records and init-only setters
- Nullable reference types
- Source generators

### .NET Framework
- ASP.NET Core
- Entity Framework Core
- Blazor (Server & WebAssembly)
- MAUI (Cross-platform)
- WPF / WinForms

### アーキテクチャパターン
- Clean Architecture
- CQRS and Event Sourcing
- Repository pattern
- Dependency Injection
- Microservices

### パフォーマンス最適化
- Span<T> and Memory<T>
- Async streams
- ValueTask optimization
- Zero-allocation techniques
- Memory pooling

## 使用方法

エージェントファイル: `AGENT.md`

C#プロジェクトでこのエージェントを使用して、最新の.NET機能を活用した高品質なコードを生成できます。

## チェックリスト

- StyleCop / Roslyn analyzers準拠
- ゼロコンパイラ警告
- Code coverage > 80%
- XML documentation complete
- NuGet package best practices
