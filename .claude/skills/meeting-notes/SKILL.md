# Meeting Notes Skill

会議の議事録を構造化して生成するスキルです。

## 概要

会議の音声文字起こしやメモから、整理された議事録を自動生成します。

## 主な機能

- **アジェンダ抽出**: 会議のトピックを整理
- **決定事項**: 合意された内容を明確化
- **アクションアイテム**: タスクと担当者を抽出
- **参加者リスト**: 出席者の記録
- **要約生成**: エグゼクティブサマリー
- **フォローアップ**: 次回までのTO DO

## 使用方法

```
以下の会議メモから議事録を作成：

[会議の文字起こしやメモ]

含める内容:
- 決定事項
- アクションアイテム（担当者と期限）
- 次回会議の予定
```
