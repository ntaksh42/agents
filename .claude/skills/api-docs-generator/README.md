# API Documentation Generator Skill

APIドキュメントを自動生成するClaude Code スキルです。

## 概要

ソースコードから美しく、詳細で、インタラクティブなAPIドキュメントを自動生成します。OpenAPI/Swagger、Markdown、HTML、Postman Collection等に対応します。

## 主な機能

- ✅ OpenAPI/Swagger 3.0 仕様生成
- ✅ Markdown形式ドキュメント
- ✅ インタラクティブSwagger UI
- ✅ Postman Collection
- ✅ GraphQL Schema ドキュメント
- ✅ 多言語コード例（JS, Python, cURL等）
- ✅ 認証・エラーコード・レート制限情報

## クイックスタート

```
このAPIエンドポイントのドキュメントを生成：

GET /api/users

形式: OpenAPI 3.0
```

## サポートフレームワーク

- REST: Express, FastAPI, Flask, Spring Boot, ASP.NET Core
- GraphQL: Apollo, GraphQL Yoga
