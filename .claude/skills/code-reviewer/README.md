# Code Reviewer Skill

プロフェッショナルなコードレビューを自動化するClaude Code スキルです。

## 概要

セキュリティ、パフォーマンス、保守性、ベストプラクティスの観点から包括的なコードレビューを実施し、具体的な改善提案を提供します。

## 主な機能

- ✅ セキュリティ脆弱性の検出（OWASP Top 10）
- ✅ パフォーマンスボトルネックの特定
- ✅ コード品質とベストプラクティスの評価
- ✅ アーキテクチャとデザインパターンの分析
- ✅ 具体的な修正コード例の提供
- ✅ 重大度レベル別の問題分類
- ✅ 50以上のプログラミング言語対応

## クイックスタート

```
このファイルをレビューしてください：
src/api/users.py
```

または

```
以下のPRをレビュー：
変更内容: JWT認証の実装
重点: セキュリティとパフォーマンス
```

## レビュー観点

1. **セキュリティ**: SQLインジェクション、XSS、認証・認可、CSRF等
2. **パフォーマンス**: N+1クエリ、アルゴリズム複雑度、メモリリーク等
3. **コード品質**: DRY原則、単一責任、命名規則、複雑度等
4. **アーキテクチャ**: SOLID原則、デザインパターン、レイヤー分離等
5. **テスト**: カバレッジ、品質、エッジケース等
6. **エラーハンドリング**: 例外処理、ロギング、リソース管理等
