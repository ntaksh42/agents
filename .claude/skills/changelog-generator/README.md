# Changelog Generator Skill

Gitコミット履歴から美しいChangelogを自動生成するClaude Code スキルです。

## 主な機能

- ✅ Keep a Changelog / Conventional Commits形式
- ✅ 自動カテゴリ分類（feat, fix, docs等）
- ✅ Breaking Changes検出
- ✅ GitHub Issue/PRリンク自動生成
- ✅ Contributors リスト
- ✅ セマンティックバージョニング

## クイックスタート

```
Gitコミット履歴からChangelogを生成：
形式: Keep a Changelog
```
