# セキュリティ観点

以下のセキュリティ観点も追加で確認してください：

## OWASP Top 10
- インジェクション（SQL, コマンド, LDAP）
- 認証の不備
- 機密データの露出
- XXE（XML外部エンティティ）
- アクセス制御の不備
- セキュリティ設定のミス
- XSS（クロスサイトスクリプティング）
- 安全でないデシリアライゼーション
- 既知の脆弱性を持つコンポーネントの使用
- 不十分なロギングとモニタリング

## 入力検証
- ユーザー入力のサニタイズ
- バウンダリチェック（配列、文字列長）
- 型の安全性

## 認証・認可
- 認証トークンの適切な管理
- 権限チェックの漏れ
- セッション管理

## 機密情報
- ハードコードされたシークレット
- ログへの機密情報出力
- エラーメッセージによる情報漏洩
