# Templates

C#/C++プロジェクトで再利用可能なテンプレート集

## 利用可能なテンプレート

### CMake テンプレート (`cmake/`)
- 基本的なC++プロジェクト構成
- ライブラリとの連携設定
- テスト構成

### C# プロジェクトテンプレート (`csharp-project/`)
- WPFアプリケーション
- コンソールアプリケーション
- クラスライブラリ

### C++ プロジェクトテンプレート (`cpp-project/`)
- ヘッダーオンリーライブラリ
- 静的/動的ライブラリ
- 実行ファイルプロジェクト
